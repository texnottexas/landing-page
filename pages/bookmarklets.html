Top War — Bookmarklet Launcher

Inventory + Enigma Beast extractors for S2864 players. Output → clipboard. Paste into 2864tw.com.

Desktop (Chrome / Safari / Firefox)

Drag the green button to your bookmarks bar. Then visit the game and click the bookmark.

iOS Safari

Apple blocks drag-to-bookmarks on iOS. Add manually:

  1. Tap the Share icon → Add Bookmark (any page works) → Save
  2. Open Bookmarks → Edit → tap the new bookmark
  3. Tap "Copy" below to copy the URL, then paste into bookmark URL field
  4. Save
  5. Open the game in Safari → tap the bookmark while the relevant panel is open

Android Chrome

Same idea — bookmark any page, then edit and replace its URL.

  1. Open this page in Chrome
  2. Tap menu (⋮) → Add to bookmarks (any page works)
  3. Tap menu (⋮) → Bookmarks → tap and hold the new bookmark → Edit
  4. Tap "Copy" below, paste into the URL field, save
  5. To run: open the game, type the bookmark name into the URL bar, tap the autocomplete suggestion (Chrome strips javascript: when typed manually — picking via autocomplete preserves it)

Usage

Inventory

  1. Open game → tap Bag (bottom-right)
  2. Tap any tab (Item / Unit / Decor / Hero / CPNT)
  3. Tap bookmarklet → JSON copied to clipboard

Enigma Beasts

  1. Open game → Enigma Beast section → tap the panel that lists every owned beast (the one with quality/faction/type filters)
  2. Wait for list to fully load
  3. Tap bookmarklet → JSON copied (Q3+ unplaced only — placed beasts already in armory report)

Note: only run when the relevant panel is open. Bookmarklet reads scene-graph state, not server data.
