Buff system — per-row 521xxx family live

Each awakening level grants 4 buffs: 3 per-row variants (front / mid / back) from one specialty family, plus 1 global buff for the hero's army type. Hero scaling caps at +900 per row + a +3,600 global buff at the 40-level peak.

Awakening Level Cost — full 40-level ladder live

From hero_awakening_level (40 rows). Each level needs 100 EXP, gained at 20 EXP per pull → 5 pulls per level. Gold Shard and ES Skill columns are per-hero items — the actual item IDs differ per hero (see hero_awakening_hero mapping). Per-pull cost shown with the per-level total (×5) in parentheses. Cost tiers visible at a glance: Lv.1–9 = 2|0, Lv.10–19 = 4|2, Lv.20–29 = 6|4, Lv.30–40 = 10|8 (Gold Shard | ES Skill per pull). To max one hero from 0 → 40, you need roughly 1,000 Hero Shards + 700 ES Skill chips of that hero.

How awakening progression works live

Hero level — 40 ranks

Each awakened hero has 40 levels of awakening progression. Each level grants ~2000 power (capping at 80,000 power total). Item cost ramps: 2|0 shards/level for Lv.1–9, 4|2 for Lv.10–19, up to 10|8 for Lv.30–40.

Skill level — 10 ranks each

Each of the 4 awakening skills has its own Lv.1–10 ladder. hero_awakening_skill rows tag each level's required buff (the 21x101 family) and the Exclusive-7 alternate (the 21x1101 family) — proof that Exclusive Skill Lv.7 swaps in a more potent buff.

maxUpgrade — 7,200 rows

The hero_awakening_maxUpgrade table (7,200 rows, 18 heroes × 4 skills × 100 sub-levels) holds the late-game progression. Each row stores 2 buff IDs (Army/Navy/Air-specific from the 521xxx family) and a per-row power gain.

Awakening Choice Chest cadence

Each Phase = 7-day box (item_box_countdown: 604800). 1 reward delivered immediately; remaining 5 unlock on a 7-day cadence; 6 total per phase. Backlog accumulates if not collected.
