# Titan Canyon

A minimal landing page for Titan Canyon (Server 2864) — built with vanilla HTML/CSS/JS and hosted on GitHub Pages.

## Links

- **Signups** — [Google Form](https://forms.gle/2UaSmMxZEfVfhh617)
- **Rosters** — [titan.2864tw.com](https://titan.2864tw.com)

## Features

- Dark / Light mode toggle with localStorage persistence
- Responsive, mobile-friendly layout
- Image modal viewer
- No dependencies — pure HTML, CSS, and JavaScript

## Deployment

This site is automatically deployed via GitHub Pages from the `main` branch.
