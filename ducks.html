What is Claude Code?

Claude Code is an AI coding agent by Anthropic that lives right inside VS Code. It can read your code, write new code, run commands, search the web, manage files, and even create git commits — all through natural language conversation.

The VS Code Extension

Claude Code runs as a native VS Code extension, meaning it's deeply integrated into your editor — not a separate app or browser tab. It sees what you see.

Claude Code's Superpowers (Tools)

Claude Code has a toolbelt of powerful capabilities it can use autonomously:

  • Read & Edit files — Opens, understands, and modifies any file in your project
  • Run terminal commands — Executes builds, tests, installs, git, and more
  • Search & Navigate — Finds files by pattern, searches code content with regex
  • Web Search & Fetch — Looks up docs, APIs, and references on the internet
  • Multi-agent delegation — Spawns sub-agents to research or work in parallel
  • Memory & Context — Remembers your preferences and project details across sessions
Imagine having six ducks, each specialized in a different task, and they all work together. That's Claude Code's tool system.

How a Typical Flow Works

You describe what you want in plain English (or Duck): "Add dark mode to the settings page"
Claude reads your codebase — finds relevant files, understands the architecture, checks existing patterns
Claude writes a plan — proposes what files to change and why, so you can approve before any edits
Claude edits the code — makes precise, minimal changes across multiple files simultaneously
Claude runs tests — executes your test suite to make sure nothing broke
You review & approve — Claude never pushes to production without your say-so. You're always the boss duck. 🦆👑
You quack the order, Claude does the swimming. You still steer.
